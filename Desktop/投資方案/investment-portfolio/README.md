# 宜蘭房地產投資方案網站

這是一個展示宜蘭地區房地產投資方案的靜態網站。

## 功能特點

- 響應式設計，支援各種裝置
- 現代化的使用者介面
- 平滑的滾動效果
- 圖片優化載入

## 部署到 GitHub Pages

1. 在 GitHub 創建新的儲存庫
2. 將所有檔案推送到儲存庫：
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin <你的儲存庫URL>
   git push -u origin main
   ```
3. 在儲存庫設定中啟用 GitHub Pages：
   - 前往儲存庫的 Settings > Pages
   - 在 Source 選擇 main 分支
   - 點擊 Save

## 本地開發

直接在瀏覽器中開啟 `index.html` 即可預覽網站。
